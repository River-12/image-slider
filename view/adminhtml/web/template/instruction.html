1. CMS Page/Static Block
    • Method 1: Click on Insert Widget icon. Inside the popup, select "Image Slider Widget" and then select the slider from the dropdown.
    • Method 2: {{widget type="Riverstone\ImageSlider\Block\Widget\Slider" slider_identifier=""}}

  2. Template file (.phtml)
    <?php echo $block->getLayout()->createBlock("Riverstone\ImageSlider\Block\Widget\Slider")->setSliderIdentifier("")->toHtml();?>
